# Snake Game - Python Turtle

This is a simple Snake Game project built with Python's built-in `turtle` module.

The project was created as a practice project to understand object-oriented programming, game loops, keyboard events, collision detection, and file-based project structure in Python.

## Features

- Snake movement
- Keyboard controls
- Food spawning at random positions
- Snake growth after eating food
- Score tracking
- Wall collision detection
- Tail collision detection
- Game Over screen

## Project Structure

```text
snake-game-python/
│
